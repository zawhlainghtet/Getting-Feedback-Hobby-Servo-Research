          This research focuses on learning and applying industrial robotics concepts to students with a low-cost, safe, and practical solution. By modifying cheap hobby servos into feedback-capable digital or analog hobby servos, the system reduces electronic waste by the burnt servos, improves learning outcomes such as usage of feedback data and filter system, and learning the concepts between educational robotics and real-world industrial servo systems. This modification provides an effective teaching robotics fundamentals while being easy to implement. 
          Using this modification and implementation, learners can add and study control  and analysis system. The RC low-pass filter represents a first-order system. It can be modeled using, DFT, Z-transform (discrete). Filtering improves signal quality before control or analysis. Mapping converts sensor signals into angles that can be what users wanted.
